라디오 버튼

        라디오 버튼은 같은 name 을 주어야 하나의 그룹으로 묶인다.
        라디오 버튼은 하나의 값만 선택하게 할 때 사용한다.(다중선택 X)
        기본값으로 체크가 되어있게 하려면, checked 속성 부여
        제출(submit)시, check 된 input 태그의 value 가 전달된다.
        value 가 없으면, on 이라는 값으로 전달된다.






체크박스

        체크박스는, 다중선택을 할 때 사용한다. 
        체크박스 선택 시 해당 태그의 value 가 전달된다. 
        name 값은 똑같이 주면 된다. (어차피 전부 넘어간다.)
        기본적으로 체크될 요소들을 지정할 수 있다. (checked)
